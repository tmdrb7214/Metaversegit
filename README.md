발생한 문재점 ,해결

씬이 한번만작동하고 다시하기나 ,마을로 돌아갔을경우 멈추게되었다.
Time.timeScale =1 ;  타임스케일을 씬시작시에 1로 넣어 해결되었다 .

플레이어가 collider 충돌도 무시하고 멈추지않고 튕겨져나가는 현상이 발생했다 .
